OpenGL：变换

发表于2024-09-24|笔记
|字数总计:1.2k|阅读时长:4分钟|阅读量:

之前的图像都是静态的，我们可以尝试在每一帧改变物体的顶点并且重新配置缓冲区从而实现动态，但这太繁琐了，因此我们需要通过矩阵对象来实现变换

1. 向量

向量（Vector）：一个有方向（Direction）和大小（Magnitude）的量，向量的每一项叫做分量，默认向量的起点是原点，因此只需要指定向量的终点就可以指示一个方向

向量运算描述
标量运算一个向量加/减/乘/除一个标量，相当于对向量的每个分量分别进行该运算
取反将向量的每个分量取反
加减两个向量的对应分量进行加减
长度对分量进行平方和后开根号
归一每个分量除以向量的长度得到单位向量
点积将对应分量逐个相乘
叉积生成一个正交于两个输入向量的第三个向量

2. 矩阵

矩阵（Matrix）：一个二维数组，矩阵中每一项叫做矩阵的元素（element），矩阵可以通过(i, j)进行索引，i是行，j是列，矩阵的行数和列数分别叫做矩阵的维度（dimension）

矩阵运算描述
加减两个矩阵对应位置的元素进行加减
数乘矩阵的每个元素乘以一个标量
相乘新矩阵每一个元素是对应行和对应列的线性组合

向量可以看作是一个Nx1的矩阵

3. 变换

在OpenGL中，矩阵通常是4x4的，向量通常是4x1的，并且使用向量左乘矩阵实现变换

3.1 缩放

缩放（Scalling）：对向量的不同分量的长度进行缩放（倍增或倍减）

缩放矩阵：除了矩阵左对角线上元素，其他元素的值都为0

(4,4)位置的元素值只能是1，因为在3D空间中缩放w分量是无意义的

3.2 位移

位移（Translation）：在原始向量的基础上加上另一个向量从而获得一个在不同位置的新向量

位移矩阵：除了矩阵左对角线上元素和第四列的元素，其他元素的值都为0

3.3 旋转

旋转（Rotation）：需要指定一个旋转轴和一个旋转角度，轴是一个向量，角度是一个标量（弧度制或角度制）

旋转矩阵

3.4 组合变换

假如缩放矩阵是A，位移矩阵是B，旋转矩阵是C，并且对向量v先进行缩放再进行位移最后进行旋转，可以得到最后的向量是u = A * B * C * v，再根据矩阵乘法的结合律，我们可以将它们合并成一个矩阵D = A * B * C，因此u = D * v

矩阵乘法是不遵守交换律的，因此不能改变矩阵的先后顺序！

4. GLM

GLM（OpenGL Mathematics）：专门为OpenGL量身定做的数学库，它提供了许多数学函数和数据类型，用于处理图形学中的各种数学运算

3个头文件数据类型向量运算（v1和v2是vec3类型）矩阵运算（m1和m2是mat4类型）变换
